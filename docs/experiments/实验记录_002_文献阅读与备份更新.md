## 📊 核心知识点总结

### 1. 关联函数理论

**关键公式**:
```
Π_f(σ) = (1 / N_Df) * Σ_l Π_{i∈f} S_i
⟨Π_f⟩_R = (2x - 1)^k  (二元合金，x=0.5 时为 0)
```

**物理意义**:
- 一阶关联：平均浓度
- 二阶关联：原子对分布（短程序）
- 三阶及以上：多体关联

### 2. SQS 方法

**核心思想**:
- 设计特殊周期结构 S
- 使关联函数 Π_f(S) 最佳匹配随机合金 ⟨Π_f⟩_R
- 用有限尺寸超胞模拟无限大随机合金

**推荐结构**:
- **FCC 晶格**: SQS-8 (A₂B₃A₂B₁ [113])
- **BCC 晶格**: SQS-2 (CsCl 型)

### 3. SSOS 方法

**创新点**:
- 用多个小结构的加权平均代替单个大超胞
- ⟨f⟩_R ≈ Σ w_i * f(σ_i^SSOS)
- 计算成本降低 10-40 倍

**应用实例**:
- **高熵合金**: SSOS-3×5 (3 个 5 原子结构)
- **尖晶石**: SSOS-2×28 (2 个 28 原子结构)

### 4. 不可约构型生成

**算法流程**:
1. 搜索空间群操作 (Spglib)
2. 构建等价原子矩阵
3. 枚举所有组合构型
4. 整数表示加速去重

**性能优势**:
- 时间复杂度：O(N_irred)
- 支持任意超胞（包括非对角）
- 支持多元系统

## 🎯 物理理解要点

### 为什么需要 SQS/SSOS？

**问题**:
- 第一性原理计算成本高 O(N³)
- 随机合金需要大超胞（>100 原子）
- 蒙特卡洛需要大量采样

**解决方案**:
- SQS: 单个小超胞（8-36 原子）
- SSOS: 多个小结构加权平均（2-3 个）

**物理本质**:
- 固体性质由短程相互作用主导
- 关联函数提供统计描述的自然基
- 对称性保证小样本的代表性

### 结构弛豫的重要性

**尺寸失配合金**（如 GaSbAs）:
- 未弛豫：混合焓误差 >100 meV
- 弛豫后：混合焓误差 <10 meV

**原因**:
- 原子半径差异导致局域晶格畸变
- 键长分布展宽（不是单一平均值）
- 影响电子结构和能隙
