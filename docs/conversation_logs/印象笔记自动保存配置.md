## ⚙️ 配置说明

### 自动保存触发条件

以下情况会**自动保存**到印象笔记：

1. ✅ **完成重要任务后**
   - 文献阅读与分析
   - 代码模块开发
   - 系统配置更新
   - 问题排查与解决

2. ✅ **创建关键文档后**
   - 文献笔记
   - 实验记录
   - 技术总结
   - 对话记录

3. ✅ **Git 提交后**
   - 有实质性内容的提交
   - 新功能/模块完成
   - 重要 bug 修复

4. ✅ **用户明确要求时**
   - "保存这次对话"
   - "同步到印象笔记"

## 📝 保存格式

### 笔记标题格式
```
对话记录 XXX - 主题（YYYY-MM-DD）
```

**示例**:
- `对话记录 001 - 文献阅读与备份更新（2026-03-01）`
- `对话记录 002 - Fortran 代码分析（2026-03-02）`

### 笔记内容结构

```markdown
# 对话记录 XXX - 主题

**日期**: YYYY-MM-DD  
**主题**: 简短描述  
**状态**: ✅ 完成 / 🔄 进行中 / ❌ 失败

## 任务目标
- [x] 目标 1
- [x] 目标 2

## 执行内容
### 1. 任务名称
- 操作：具体操作
- 结果：✅ 成功 / ❌ 失败

## 关键成果
### 创建的文件
- 文件 1
- 文件 2

### Git 提交
- commit_hash - 提交信息

## 问题与解决
- 问题：描述
- 解决：方案

## 下一步计划
- [ ] 待办事项

## 统计数据
- 文档数：X 个
- 代码行数：X 行
- Git 提交：X 次

---
**记录时间**: YYYY-MM-DD HH:MM  
**GitHub**: 仓库链接
```

## 🔄 保存流程

### 自动保存流程

1. **检测重要对话** → AI 判断
2. **整理对话内容** → 提取关键信息

4. **保存到印象笔记** → MCP 调用
5. **本地备份** → Markdown 文件
6. **Git 提交** → 版本控制
7. **确认保存** → 返回结果
