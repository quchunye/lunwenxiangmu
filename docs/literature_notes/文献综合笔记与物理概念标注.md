## 🔑 核心物理概念

### 1. 关联函数 (Correlation Functions)

#### 定义
对于合金构型 σ，在晶格位置 i 定义赝自旋变量 Si：
- **二元合金**: Si = -1 (A 原子), Si = +1 (B 原子)
- **多元合金**: Si = 0, 1, 2, ..., M-1 (M 种元素)

#### 多体关联函数
```
Π_f(σ) = (1 / N_Df) Σ_l Π_{i∈f} S_i
```
其中：
- f = (k, m) 表示图形类型（k 个点，m 近邻距离）
- l 遍历所有对称相关的位置
- Df 是每个位置的图形数量

#### 完美随机合金的关联函数
对于浓度为 x 的随机合金：
```
⟨Π_f⟩_R = (2x - 1)^k  (二元合金)
⟨Π_f⟩_R = Π_i x_i^{n_i}  (多元合金，x_i 是第 i 种元素浓度)
```

**物理意义**: 
- 一阶关联函数 (k=1): 平均浓度
- 二阶关联函数 (k=2): 原子对分布函数
- 三阶关联函数 (k=3): 三角形构型分布
- 更高阶：描述更复杂的多体关联

---

### 2. 特殊准随机结构 (SQS)

#### 核心思想
设计特殊的周期性结构 S，使其关联函数 Π_f(S) 最佳匹配随机合金的系综平均 ⟨Π_f⟩_R：
```
minimize: Σ_f w_f |Π_f(S) - ⟨Π_f⟩_R|^2
```

#### SQS-8 (FCC 晶格，x=0.5)
- **结构**: A₂B₃A₂B₁ 沿 [113] 方向的超晶格
- **优点**: 前 2 近邻关联函数完美匹配随机合金
- **应用**: 仅需 8 个原子即可准确模拟无限大随机合金

#### 物理性质计算
物理性质 P 的展开：
```
⟨P⟩ = Σ_f D_f ⟨Π_f⟩ J_f
```
其中 J_f 是有效团簇相互作用 (ECI)

**关键洞察**: 
- 近邻相互作用贡献最大（层次化原理）
- 结构弛豫对尺寸失配合金至关重要
- SQS 方法避免了团簇展开的截断误差

### 3. 小有序结构集 (SSOS)

#### 核心创新
用多个小结构的加权平均代替单个大超胞：
```
⟨f⟩_R ≈ Σ_{i=1}^n w_i f(σ_i^SSOS)
```
约束条件：
- w_i > 0
- Σ w_i = 1

#### 两步优化过程
1. **内层优化**: 对给定结构集，用最小二乘法求最优权重 w_i
2. **外层优化**: 从候选结构池中选择最佳结构组合

#### 优势对比

| 方法 | 计算成本 | 精度 | 适用范围 |
|------|---------|------|---------|
| SQS-168 | O(N³) 高 | 基准 | 二元、三元合金 |
| SSOS-2×28 | O(2×28³) 低 | 与 SQS-168 相当 | 多元合金 |
| **加速比** | **>10 倍** | - | - |

#### 应用实例
**高熵合金 (BCC A₁B₁C₁D₁E₁)**:
- SSOS-3×5: 3 个 5 原子结构
- 完美匹配前 2 近邻的 20 个关联函数
- 等效于 SQS-125，但计算速度快 40 倍

### 4. 不可约位置占据构型

#### 问题背景
对于 K 个位置、N 种原子、化学计量比 {k₁, k₂, ..., k_N}：
```
总构型数 = K! / (k₁! k₂! ... k_N!)
```
组合爆炸：K=20, k₁=k₂=10 → 184,756 种构型

#### 对称性约化
许多构型在几何上是等价的（通过空间群操作相关联）。

**目标**: 生成所有不等价的不可约构型

#### 算法流程

**步骤 1: 搜索空间群操作**
- 使用 Spglib 库
- 旋转矩阵 W (3×3 整数矩阵，|det(W)|=1)
- 平移向量 ω (3×1 小数向量)
- 操作：x' = Wx + ω

**步骤 2: 构建等价原子矩阵**
对每个空间群操作，记录原子标号的置换：
```
等价矩阵行：[permuted_atom_indices]
```

**步骤 3: 枚举所有组合构型**
- 二元合金：组合数 C(k₁, K)
- 多元合金：多重组合数分解为多个二元组合

**步骤 4: 整数表示加速去重**
将构型映射为连续整数：
```
index = f(configuration)
```
优势：
- 线性时间复杂度 O(N_irred)
- 快速查找和比较

#### 性能对比

| 软件 | 支持非对角超胞 | 多元系统 | 时间复杂度 |
|------|--------------|---------|-----------|
| SOD | ❌ | 仅二元 | 指数增长 |
| enumlib | ✅ | ✅ | O(N_irred) |
| supercell | ❌ | ✅ | O(N_irred), 快 10 倍 |
| **disorder (本文)** | ✅ | ✅ | **O(N_irred), 最快** |

## 🔬 物理意义理解指南

### 1. 为什么需要 SQS/SSOS？

**问题**: 
- 第一性原理计算成本高（O(N³) 标度）
- 随机合金需要大超胞（>100 原子）才能描述统计分布
- 蒙特卡洛方法需要大量采样（~1000 个构型）

**解决方案**:
- SQS: 用单个小超胞（8-36 原子）完美匹配关联函数
- SSOS: 用多个小结构的加权平均（2-3 个结构）

**物理本质**:
- 固体物理性质由短程相互作用主导
- 关联函数提供了统计描述的自然基
- 对称性保证了小样本的代表性

### 2. 关联函数的层次结构

```
重要性排序:
1. 一阶关联 (浓度) - 必须匹配
2. 二阶关联 (近邻对) - 最重要
3. 三阶关联 (三角形) - 次重要
4. 四阶及以上 - 可忽略（通常）
```

**例外情况**:
- 长程库仑相互作用（离子晶体）
- 磁性系统（长程磁关联）
- 需要更多近邻壳层匹配

### 3. 结构弛豫的重要性

**尺寸失配合金**（如 GaSbAs）:
- 未弛豫：混合焓误差 >100 meV
- 弛豫后：混合焓误差 <10 meV

**物理原因**:
- 原子半径差异导致局域晶格畸变
- 键长分布展宽（不是单一平均值）
- 影响电子结构和能隙
