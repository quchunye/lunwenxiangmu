## 📐 核心公式速查

### 1. 关联函数 (Correlation Functions)

```python
# 二元合金 (Ising 模型)
S_i = -1  # A 原子
S_i = +1  # B 原子

# k 体关联函数
Π_f(σ) = (1 / N_Df) * Σ_l Π_{i∈f} S_i

# 完美随机合金 (浓度 x)
⟨Π_f⟩_R = (2x - 1)^k  # k 是团簇的点数

# 示例：x=0.5 时
⟨Π_1⟩ = 0      # 一阶（浓度）
⟨Π_2⟩ = 0      # 二阶（配对）
⟨Π_3⟩ = 0      # 三阶（三角形）
```

### 2. 物理性质展开

```python
# 团簇展开公式
⟨P⟩ = Σ_f D_f * ⟨Π_f⟩ * J_f

# 其中:
# D_f - 简并度（每个位置的图形数）
# J_f - 有效团簇相互作用 (ECI)
# ⟨Π_f⟩ - 关联函数的系综平均
```

### 3. SQS 优化目标

```python
# 最小化关联函数误差
E = Σ_{f} w_f * |Π_f(SQS) - ⟨Π_f⟩_R|^2

# 权重选择（层次化原理）
w_f = 1.0    # 第 1 近邻
w_f = 0.5    # 第 2 近邻
w_f = 0.25   # 第 3 近邻
# ... 指数衰减
```

### 4. SSOS 加权平均

```python
# 物理性质计算
⟨f⟩_R ≈ Σ_{i=1}^n w_i * f(σ_i^SSOS)

# 约束条件
w_i > 0
Σ w_i = 1

# 最小二乘优化
min_w Σ_α Σ_s [Σ_i w_i * Π_α^s(σ_i) - ⟨Π_α^s⟩_R]^2
```

### 5. 构型计数

```python
# 总构型数（多重组合数）
N_total = K! / (k₁! * k₂! * ... * k_N!)

# 二元合金
N_total = C(k₁, K) = K! / (k₁! * k₂!)

# 示例: K=20, k₁=k₂=10
N_total = 184,756  # 组合爆炸！

# 不可约构型数（对称性约化）
N_irreducible = N_total / N_symmetry_ops

# 示例：FCC 超胞，48 个空间群操作
N_irreducible ≈ 184756 / 48 ≈ 3,849
```

## 🔍 对称性操作速查

### 空间群操作表示

```python
# 操作：x' = W*x + ω
# W: 3×3 旋转矩阵（整数）
# ω: 3×1 平移向量（小数）

# 示例：FCC 的对称操作
W = [[0, 1, 0],
     [1, 0, 0],
     [0, 0, 1]]  # 绕 z 轴旋转 90°

ω = [0.0, 0.0, 0.0]  # 无平移
```

### 等价原子矩阵

```python
# 原子标号：[0, 1, 2, 3, 4, 5]
# 对称操作后的标号：[1, 0, 3, 2, 5, 4]

等价矩阵行 = [1, 0, 3, 2, 5, 4]

# 构型等价性判断
config1 = [A, B, A, B, A, B]
config2 = [B, A, B, A, B, A]

# 应用对称操作
config2_transformed = apply_permutation(config2, [1, 0, 3, 2, 5, 4])
# = [A, B, A, B, A, B] = config1

# 结论：config1 和 config2 等价
```

## 📊 物理量计算

### 混合焓 (Mixing Enthalpy)

```python
# 定义
ΔH(x) = E_total(alloy) - x*E_A - (1-x)*E_B

# 单位转换
1 eV/atom = 96.485 kJ/mol
1 meV/atom = 0.096485 kJ/mol

# 典型值范围
# 半导体合金：10-100 meV/atom
# 金属合金：1-50 meV/atom
```

### 光学弯曲系数 (Optical Bowing)

```python
# 定义
b = 4 * [Eg(x=0.5) - 0.5*Eg(A) - 0.5*Eg(B)]

# 单位：eV

# 典型值
# GaAsP: b ≈ 0.2 eV
# AlGaAs: b ≈ 0.1 eV
# GaInP: b ≈ 0.7 eV
```

### 形成能 (Formation Energy)

```python
# 定义
E_form = E_total(alloy) - Σ_i x_i * E_i(pure)

# 稳定性判据
E_form < 0: 热力学稳定
E_form > 0: 亚稳/不稳定

# 高熵合金筛选
E_instability = E_form - E_convex_hull
E_instability < 0.1 eV/atom: 可能形成单相固溶体
```

## 🧮 组合数学公式

### 二项式系数

```python
C(n, k) = n! / (k! * (n-k)!)

# 帕斯卡恒等式
C(n, k) = C(n-1, k-1) + C(n-1, k)

# 快速计算（避免溢出）
C(n, k) = Π_{i=1}^k (n+1-i) / i
```

### 多重组合数

```python
# N 种原子，K 个位置
N_config = K! / (k₁! * k₂! * ... * k_N!)

# 递推计算
N_config = C(k₁, K) * C(k₂, K-k₁) * C(k₃, K-k₁-k₂) * ...

# 示例：K=6, k₁=2, k₂=3, k₃=1
N_config = C(2,6) * C(3,4) * C(1,1)
         = 15 * 4 * 1 = 60
```

### 构型到整数的映射

```python
# 二元合金，K 个位置，k₁个 A 原子
# 构型 a = [a₁, a₂, ..., a_k₁] (原子位置索引，升序)

# 配置索引计算
index = C(k₁, K) - Σ_{j=1}^{k₁} E[a_j, j]

# 其中 E 是预计算的组合数矩阵
E[i, j] = Σ_{m=1}^i C(k₁-j, K-j+m-2)

# 性质：index 是连续整数 [0, N_config-1]
```

## 💡 物理直觉培养

### 1. 关联函数的物理图像

```
随机合金 (x=0.5):
位置： 0   1   2   3   4   5
原子： A   B   B   A   B   A
自旋：-1  +1  +1  -1  +1  -1

第 1 近邻配对 (距离=a):
(0,1): -1*+1 = -1
(1,2): +1*+1 = +1
(2,3): +1*-1 = -1
...
平均：Π_2,1 = <Si*Sj> ≈ 0  (随机)

有序结构 (CuAu):
A B A B A B  →  Π_2,1 = -1  (完全有序)
A A A A A A  →  Π_2,1 = +1  (完全偏聚)
```

### 2. SQS 的"准随机"特性

```
真实随机合金:
- 无限大体系
- 关联函数：⟨Π_f⟩_R = 0 (x=0.5)
- 无法直接计算

SQS-8:
- 有限周期超胞（8 原子）
- 关联函数：Π_f(SQS) ≈ 0
- 可以进行第一性原理计算

关键：用周期性误差换取有限尺寸
```

### 3. SSOS 的"分而治之"思想

```
传统方法:
1 个大超胞 (125 原子) → O(125³) = 1,953,125

SSOS 方法:
3 个小超胞 (5 原子) → 3 * O(5³) = 375

加速比：1,953,125 / 375 ≈ 5,200 倍！

物理本质：用多个小样本的统计平均代替大样本
```
