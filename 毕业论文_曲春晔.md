# 基于人工智能编程工具的无序合金模拟智能化系统设计与实现

---

## 摘要

无序合金是一类重要的工程材料，广泛应用于航空航天、能源、催化等领域。准确预测无序合金的热力学性质对于材料设计具有重要意义。然而，传统的第一性原理计算方法难以直接处理无序体系，需要构建特殊的结构模型来模拟无序状态。特殊准随机结构（SQS）和有序结构小集合（SSOS）是两种常用的无序合金建模方法，但现有工具存在操作复杂、流程割裂、自动化程度低等问题。

本文设计并实现了一套基于人工智能编程工具的无序合金模拟智能化系统。系统采用模块化架构，包含四个核心模块：超胞构建模块、不可约构型生成模块、关联函数计算模块和SSOS搜索模块。超胞构建模块基于对称性分析和Delaunay三角化优化，自动生成所有可能的超胞结构；不可约构型生成模块利用空间群对称性消除等价构型，显著减少计算量；关联函数计算模块采用团簇展开方法，高效计算各构型的关联函数；SSOS搜索模块基于随机化正交匹配追踪算法，从大量候选构型中筛选出最优结构组合。

本系统的主要创新点包括：（1）提出了完整的无序合金模拟自动化工作流程；（2）实现了超胞生成的对称性去重和几何优化；（3）开发了基于Python的SSOS搜索算法；（4）设计了模块化的系统架构，便于扩展和维护。以Cu-Au二元合金为例进行了系统验证，结果表明系统能够正确生成超胞、计算关联函数并搜索得到合理的SSOS结构集合。

本系统为无序合金的第一性原理计算提供了便捷的工具支持，有望加速新型合金材料的研发进程。

**关键词**：无序合金；特殊准随机结构；有序结构小集合；团簇展开；人工智能编程

## Abstract

Disordered alloys are an important class of engineering materials with wide applications in aerospace, energy, and catalysis. Accurate prediction of thermodynamic properties of disordered alloys is crucial for materials design. However, traditional first-principles calculation methods cannot directly handle disordered systems, requiring special structural models to simulate the disordered state. Special Quasirandom Structures (SQS) and Small Set of Ordered Structures (SSOS) are two commonly used methods for modeling disordered alloys, but existing tools suffer from complex operations, fragmented workflows, and low automation.

This thesis designs and implements an intelligent system for disordered alloy simulation based on AI programming tools. The system adopts a modular architecture containing four core modules: supercell construction module, irreducible configuration generation module, correlation function calculation module, and SSOS search module. The supercell construction module automatically generates all possible supercell structures based on symmetry analysis and Delaunay triangulation optimization. The irreducible configuration generation module eliminates equivalent configurations using space group symmetry, significantly reducing computational cost. The correlation function calculation module efficiently computes correlation functions using the cluster expansion method. The SSOS search module selects the optimal structure combination from numerous candidate configurations based on the randomized orthogonal matching pursuit algorithm.

The main innovations of this system include: (1) proposing a complete automated workflow for disordered alloy simulation; (2) implementing symmetry deduplication and geometric optimization for supercell generation; (3) developing a Python-based SSOS search algorithm; (4) designing a modular system architecture for easy extension and maintenance. The system was validated using Cu-Au binary alloy as an example, demonstrating correct supercell generation, correlation function calculation, and reasonable SSOS structure set search.

This system provides convenient tool support for first-principles calculations of disordered alloys and is expected to accelerate the development of new alloy materials.

**Keywords**: Disordered Alloys; Special Quasirandom Structures; Small Set of Ordered Structures; Cluster Expansion; AI Programming

# 第一章 绪论

## 1.1 研究背景与意义

### 1.1.1 无序合金的重要性

无序合金，又称固溶体，是指两种或多种元素在原子尺度上随机混合形成的合金材料。这类材料在现代工业中具有极其重要的地位，其应用领域涵盖：

**航空航天领域**：镍基高温合金是航空发动机涡轮叶片的核心材料，其优异的高温强度和抗蠕变性能源于γ'相析出强化与固溶强化的协同作用。钛铝合金则因其低密度、高比强度的特点，成为新一代航空发动机低压涡轮叶片的候选材料。

**能源领域**：核电站结构材料需要在高温、高压、强辐射环境下长期稳定运行，锆合金和镍基合金因其良好的耐腐蚀性和抗辐照性能而被广泛采用。太阳能电池中的铜铟镓硒（CIGS）薄膜也是一种典型的无序合金材料。

**催化领域**：铂基合金催化剂在燃料电池、石油化工等领域发挥着关键作用。通过调节合金成分，可以优化催化剂的活性、选择性和稳定性。近年来，高熵合金催化剂因其独特的性能而受到广泛关注。

**功能材料领域**：形状记忆合金、储氢合金、热电材料等功能材料大多属于无序合金范畴。这些材料的性能与其原子尺度的无序程度密切相关。

### 1.1.2 无序合金模拟的挑战

尽管无序合金应用广泛，但从理论上预测其性质仍面临重大挑战：

**计算复杂性**：无序合金的原子排列具有随机性，理论上存在无穷多种可能的构型。直接采用第一性原理方法计算所有可能的构型是不现实的。

**周期性边界条件的限制**：传统的第一性原理计算软件（如VASP、Quantum ESPRESSO等）基于周期性边界条件，只能处理具有平移周期性的晶体结构。无序合金缺乏长程有序性，无法直接用周期性模型描述。

**统计平均的需求**：无序合金的热力学性质是大量微观构型的统计平均结果。要准确预测这些性质，需要构建能够代表无序状态的有限规模模型。

**构型熵的贡献**：无序合金的稳定性很大程度上来源于构型熵。如何在计算中正确计入熵的贡献，是无序合金热力学计算的关键问题。

### 1.1.3 研究意义

针对上述挑战，发展高效准确的无序合金模拟方法具有重要的科学意义和应用价值：

**理论意义**：建立无序合金结构与性质之间的定量关系，深化对无序体系物理本质的理解，推动材料计算方法学的发展。

**应用价值**：为新型合金材料的设计提供理论指导，加速材料研发进程，降低实验成本，缩短研发周期。

**方法学贡献**：发展自动化、智能化的计算工具，降低无序合金模拟的技术门槛，使更多研究者能够开展相关研究。

## 1.2 国内外研究现状

### 1.2.1 无序合金建模方法

为解决无序合金的第一性原理计算问题，研究者们发展了多种建模方法：

**特殊准随机结构（SQS）方法**：由Zunger等人于1990年提出[1]，其核心思想是构建一个有限规模的周期性结构，使其局域关联函数尽可能接近完全随机合金的关联函数。SQS方法的优点是结构规模小、计算量可控，缺点是只能近似模拟无序状态，且对于复杂成分体系难以找到最优SQS。

**相干势近似（CPA）方法**：基于平均场理论，将无序合金等效为一种有效介质[2]。CPA方法的优点是计算效率高，缺点是无法处理局域结构效应和短程有序。

**团簇展开（CE）方法**：由Sanchez等人于1984年提出[3]，将合金的能量展开为关联函数的多项式。CE方法可以高效地枚举大量构型并计算其能量，是研究合金热力学的有力工具。

**有序结构小集合（SSOS）方法**：近年来发展的一种新方法[4]，通过选择少数几个有序结构的线性组合来近似无序状态。SSOS方法结合了SQS和CE方法的优点，既能保证计算精度，又能控制计算量。

**机器学习方法**：随着人工智能技术的发展，机器学习方法开始应用于无序合金模拟[5]。通过训练神经网络或高斯过程模型来预测构型能量，可以大幅提高计算效率。

### 1.2.2 现有软件工具

目前国际上已有多个用于无序合金模拟的软件工具：

**ATAT（Alloy-Theoretic Automated Toolkit）**：由van de Walle开发的综合性合金理论计算工具包[6]，包含SQS生成（mcsqs）、团簇展开（maps）、蒙特卡洛模拟（emc2）等多个模块。ATAT是目前应用最广泛的无序合金模拟工具，但主要基于命令行操作，用户界面不够友好。

**ICET（Integrated Cluster Expansion Toolkit）**：基于Python开发的团簇展开工具包[7]，提供了丰富的API接口，便于与其他软件集成。ICET的设计理念先进，但文档相对缺乏。

**CLEASE（Cluster Expansion in Atomic Simulation Environment）**：与ASE（Atomic Simulation Environment）集成的团簇展开工具[8]，可以方便地进行结构可视化和分析。

**SQS生成器**：多种商业软件（如Materials Studio）和开源工具提供了SQS生成功能，但大多功能单一，无法完成完整的模拟流程。

### 1.2.3 存在的问题

尽管已有多种工具可用，但无序合金模拟领域仍存在以下问题：

**流程割裂**：现有工具大多只覆盖模拟流程的某个环节，用户需要在不同工具之间切换，操作繁琐且容易出错。

**自动化程度低**：超胞生成、构型枚举、关联函数计算等步骤需要大量人工干预，效率低下。

**学习成本高**：命令行工具的使用需要一定的编程基础，对非计算背景的研究者不够友好。

**缺乏智能化**：现有工具缺乏智能推荐和自动优化功能，用户需要凭借经验做出决策。

# 第二章 无序合金模拟理论基础

## 2.1 无序合金与固溶体

### 2.1.1 固溶体的分类

固溶体是指溶质原子溶入溶剂晶格中形成的单一均匀相。根据溶质原子在溶剂晶格中的位置，固溶体可分为两类：

**置换固溶体**：溶质原子取代溶剂原子占据晶格位置。例如，Cu-Au合金中，Cu原子和Au原子随机占据FCC晶格的格点位置。置换固溶体的形成条件包括：原子半径相近、晶体结构相同、电负性相近等。

**间隙固溶体**：溶质原子进入溶剂晶格的间隙位置。例如，钢中的碳原子进入铁晶格的八面体间隙位置。间隙固溶体的溶质原子通常较小，如H、B、C、N等元素。

根据溶质原子的分布特征，固溶体又可分为：

**无序固溶体**：溶质原子随机分布，各格点被不同种类原子占据的概率相同。

**有序固溶体**：溶质原子呈规则排列，形成超结构。例如，Cu-Au合金在低温下可形成Cu3Au、CuAu等有序相。

**偏聚固溶体**：同类原子倾向于聚集在一起，形成原子团簇。

本论文主要研究无序固溶体的模拟方法。

### 2.1.2 无序合金的统计描述

对于A-B二元无序合金，设A原子的浓度为c，则B原子的浓度为1-c。定义格点i的占据变量：

$$\sigma_i = \begin{cases} +1 & \text{若格点}i\text{被A原子占据} \\ -1 & \text{若格点}i\text{被B原子占据} \end{cases}$$

对于完全无序合金，各格点的占据变量统计独立，满足：

$$\langle \sigma_i \rangle = 2c - 1$$

$$\langle \sigma_i \sigma_j \rangle = (2c-1)^2 \quad (i \neq j)$$

其中，$\langle \cdot \rangle$表示系综平均。

### 2.1.3 关联函数

关联函数是描述无序合金原子排列特征的重要物理量。对于由n个格点组成的团簇α，定义其关联函数为：

$$\Pi_\alpha = \langle \prod_{i \in \alpha} \sigma_i \rangle$$

对于完全无序合金，由于各格点统计独立，有：

$$\Pi_\alpha = \prod_{i \in \alpha} \langle \sigma_i \rangle = (2c-1)^n$$

特别地，对于等原子比二元合金（c=0.5），完全无序状态的所有关联函数均为零。

## 2.2 超胞构建理论

### 2.2.1 超胞的概念

超胞（Supercell）是指将原胞沿三个晶格矢量方向扩展整数倍后形成的更大周期性单元。设原胞的晶格矢量为$\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$，超胞的晶格矢量为：

$$\begin{pmatrix} \mathbf{A}_1 \\ \mathbf{A}_2 \\ \mathbf{A}_3 \end{pmatrix} = \mathbf{S} \begin{pmatrix} \mathbf{a}_1 \\ \mathbf{a}_2 \\ \mathbf{a}_3 \end{pmatrix}$$

其中，$\mathbf{S}$为超胞变换矩阵，其元素为整数，行列式$|\mathbf{S}|$等于超胞体积与原胞体积之比。

### 2.2.2 超胞变换矩阵

超胞变换矩阵$\mathbf{S}$的一般形式为：

$$\mathbf{S} = \begin{pmatrix} s_{11} & s_{12} & s_{13} \\ s_{21} & s_{22} & s_{23} \\ s_{31} & s_{32} & s_{33} \end{pmatrix}$$

为使超胞具有合理的形状，通常要求变换矩阵满足：

1. 对角元素$s_{11}, s_{22}, s_{33}$为正整数
2. 非对角元素的绝对值小于对应的对角元素
3. 行列式$|\mathbf{S}|$等于目标体积倍数

### 2.2.3 超胞生成算法

给定目标体积倍数N，生成所有可能的超胞变换矩阵的算法如下：

**算法1：超胞生成算法**

```
输入：目标体积倍数N
输出：所有可能的超胞变换矩阵集合

1. 对N进行质因数分解：N = p1^a1 × p2^a2 × ... × pk^ak
2. 枚举所有可能的(s11, s22, s33)组合，使得s11 × s22 × s33 = N
3. 对每组(s11, s22, s33)，枚举非对角元素s12, s13, s21, s23, s31, s32
4. 检验变换矩阵是否产生有效的超胞
5. 利用对称性消除等价超胞
6. 返回所有不等价超胞变换矩阵
```

### 2.2.4 Delaunay三角化优化

为使超胞形状更加规则，采用Delaunay三角化方法对超胞进行优化。Delaunay三角化的核心思想是使超胞的三个晶格矢量尽可能正交。

优化步骤如下：

1. 将超胞晶格矢量表示为三个点的坐标
2. 构建这三个点的Delaunay三角化
3. 选择使晶格矢量夹角最接近90°的表示

经过Delaunay优化后，超胞的形状更加紧凑，有利于后续的第一性原理计算。

### 2.2.5 对称性去重

晶体的空间群对称性会导致某些不同的变换矩阵产生等价的超胞。为避免重复计算，需要利用对称性消除等价超胞。

设$\mathbf{R}$为空间群的点群操作矩阵，两个变换矩阵$\mathbf{S}_1$和$\mathbf{S}_2$等价的充要条件是存在$\mathbf{R}$使得：

$$\mathbf{S}_1 = \mathbf{R} \cdot \mathbf{S}_2$$

通过遍历所有点群操作，可以识别并消除等价超胞。

## 2.3 不可约构型与对称性

### 2.3.1 构型的定义

构型（Configuration）是指超胞中各格点被不同种类原子占据的一种具体排列方式。对于包含N个格点、支持k种原子的超胞，总共有$k^N$种可能的构型。

对于二元合金（k=2），每个格点有2种占据状态，总构型数为$2^N$。当N较大时，构型数呈指数增长，直接枚举所有构型是不现实的。

### 2.3.2 对称等价构型

晶体的空间群对称性会导致某些不同的构型实际上是等价的。具体来说，如果存在空间群操作（旋转+平移）将一个构型变换为另一个构型，则这两个构型对称等价。

设$\mathbf{R}$为旋转矩阵，$\mathbf{t}$为平移矢量，构型$\sigma$和$\sigma'$等价的充要条件是：

$$\sigma'_i = \sigma_{\mathbf{R} \cdot i + \mathbf{t}}, \quad \forall i$$

### 2.3.3 不可约构型

不可约构型（Irreducible Configuration）是指在所有对称等价构型中选择一个代表。通过只考虑不可约构型，可以大幅减少需要计算的构型数量。

不可约构型的数量与空间群的阶数成反比。对于高对称性晶体，不可约构型数可以比总构型数小几个数量级。

### 2.3.4 不可约构型生成算法

生成所有不可约构型的算法如下：

**算法2：不可约构型生成算法**

```
输入：超胞结构、空间群对称操作
输出：所有不可约构型

1. 初始化不可约构型集合为空
2. 遍历所有可能的格点占据方式
3. 对每个构型：
   a. 检查是否与已有不可约构型等价
   b. 若不等价，加入不可约构型集合
4. 返回不可约构型集合
```

实际实现中，通常采用更高效的算法，如基于群论的方法或递归生成方法。

## 2.4 团簇展开与关联函数

### 2.4.1 团簇展开理论

团簇展开（Cluster Expansion, CE）是将合金的能量展开为关联函数的多项式的方法[3]。对于二元合金，能量可以表示为：

$$E(\sigma) = \sum_\alpha V_\alpha \Pi_\alpha(\sigma)$$

其中，$\sigma$表示构型，$\alpha$遍历所有团簇，$V_\alpha$为团簇$\alpha$的有效团簇相互作用（ECI），$\Pi_\alpha(\sigma)$为构型$\sigma$在团簇$\alpha$上的关联函数。

### 2.4.2 关联函数的计算

对于给定构型$\sigma$和团簇$\alpha$，关联函数的计算方法如下：

**单点团簇**：
$$\Pi_0 = \sigma_i$$

**双点团簇**：
$$\Pi_{ij} = \sigma_i \cdot \sigma_j$$

**多点团簇**：
$$\Pi_\alpha = \prod_{i \in \alpha} \sigma_i$$

对于n元合金，采用三角函数作为正交基组[3]：

### 2.4.3 正交多项式基组

对于n元合金，采用三角函数作为正交基组：

$$\Pi_k(\sigma) = \cos\left(\frac{2\pi k \sigma}{n}\right), \quad k = 0, 1, \ldots, n-1$$

这些基函数满足正交性：

$$\sum_{\sigma=0}^{n-1} \Pi_k(\sigma) \Pi_{k'}(\sigma) = n \delta_{kk'}$$

### 2.4.4 有效团簇相互作用的确定

ECI需要通过拟合已知构型的能量来确定。设有M个已知构型，其能量分别为$E_1, E_2, \ldots, E_M$，关联函数矩阵为$\mathbf{\Pi}$，则ECI向量$\mathbf{V}$满足：

$$\mathbf{E} = \mathbf{\Pi} \cdot \mathbf{V}$$

这是一个线性方程组，可以用最小二乘法求解：

$$\mathbf{V} = (\mathbf{\Pi}^T \mathbf{\Pi})^{-1} \mathbf{\Pi}^T \mathbf{E}$$

为避免过拟合，通常采用交叉验证或正则化方法。

## 2.5 特殊准随机结构与有序结构小集合

### 2.5.1 特殊准随机结构（SQS）

SQS是一种有限规模的周期性结构，其局域关联函数尽可能接近完全随机合金的关联函数[1]。

**SQS的数学定义**：

设$\Pi_\alpha^{SQS}$为SQS在团簇$\alpha$上的关联函数，$\Pi_\alpha^{random}$为完全随机合金在团簇$\alpha$上的关联函数。SQS的目标是使：

$$\min \sum_\alpha w_\alpha |\Pi_\alpha^{SQS} - \Pi_\alpha^{random}|^2$$

其中，$w_\alpha$为团簇$\alpha$的权重，通常取团簇直径的衰减函数。

**SQS生成方法**：

SQS的生成通常采用蒙特卡洛优化方法[1,9]：

1. 随机初始化一个超胞结构
2. 随机交换不同种类原子的位置
3. 计算目标函数的变化
4. 根据Metropolis准则决定是否接受交换
5. 重复步骤2-4直到收敛

### 2.5.2 有序结构小集合（SSOS）

SSOS是一种新兴的无序合金建模方法[4]，其核心思想是用少数几个有序结构的线性组合来近似无序状态。

**SSOS的数学定义**：

设$\{\sigma_1, \sigma_2, \ldots, \sigma_M\}$为候选有序结构集合，$\{w_1, w_2, \ldots, w_M\}$为对应的权重。SSOS的目标是选择k个结构（k << M），使得它们的线性组合最接近随机合金：

$$\min \left\| \sum_{j=1}^k w_j \mathbf{\Pi}(\sigma_j) - \mathbf{\Pi}^{random} \right\|_2$$

约束条件：
$$w_j \geq 0, \quad \sum_{j=1}^k w_j = 1$$

**SSOS与SQS的比较**：

| 特性 | SQS | SSOS |
|------|-----|------|
| 结构数量 | 1个 | 多个（通常4-8个） |
| 计算精度 | 近似 | 可精确控制 |
| 适用范围 | 简单成分 | 复杂成分 |
| 计算量 | 中等 | 可调节 |

### 2.5.3 SSOS搜索算法

SSOS搜索可以转化为稀疏优化问题，采用正交匹配追踪（OMP）算法求解[13]。

**正交匹配追踪算法**：

```
输入：关联函数矩阵A，目标向量b，稀疏度k
输出：稀疏解向量w

1. 初始化：残差r = b，支撑集S = ∅
2. for i = 1 to k:
   a. 计算相关性：c = A^T · r
   b. 选择最大相关性对应的索引j
   c. 更新支撑集：S = S ∪ {j}
   d. 求解最小二乘：w_S = argmin ||A_S · w - b||
   e. 更新残差：r = b - A_S · w_S
3. 返回w
```

**随机化OMP算法**：

为避免陷入局部最优，采用随机化策略：

1. 随机选择初始结构
2. 在每一步迭代中，从相关性最大的几个候选中随机选择
3. 多次运行，选择最优结果

**关联函数表生成**：

```python
class CorrFuncTable:
    """关联函数表"""
    
    def __init__(self, n_components: int = 2):
        self.n_components = n_components
        self.table = self._generate_table()
    
    def _generate_table(self) -> np.ndarray:
        """生成三角函数基组"""
        if self.n_components == 2:
            # 二元系统: [1, -1]
            return np.array([[1.0, -1.0]])
        else:
            # 多元系统: cos(2πkσ/n)
            n_func = self.n_components - 1
            table = np.zeros((n_func, self.n_components))
            for k in range(n_func):
                for sigma in range(self.n_components):
                    table[k, sigma] = np.cos(2 * np.pi * (k+1) * sigma / self.n_components)
            return table
```

**关联函数计算**：

```python
class CorrDump:
    """关联函数计算器"""
    
    def calculate_correlations(self, structure: Structure) -> np.ndarray:
        """计算结构的所有关联函数"""
        correlations = np.zeros(len(self.clusters))
        
        for t, mcluster in enumerate(self.clusters):
            if len(mcluster.clus) == 1:
                # 单点团簇
                corr = np.mean([self.corr_func_table(0, at) 
                               for at in structure.atom_type])
            else:
                # 多点团簇
                corr = self._calc_cluster_correlation(structure, mcluster)
            correlations[t] = corr
        
        return correlations
    
    def _calc_cluster_correlation(self, structure: Structure, 
                                   cluster: MultiCluster) -> float:
        """计算单个团簇的关联函数"""
        total_corr = 0.0
        count = 0
        
        # 遍历超胞中所有团簇出现位置
        for base_idx in range(len(structure.atom_pos)):
            cluster_corr = 1.0
            valid = True
            
            # 找到团簇中各点对应的原子
            for i, clus_pos in enumerate(cluster.clus):
                atom_idx = self._find_atom(clus_pos, structure)
                if atom_idx < 0:
                    valid = False
                    break
                atom_type = structure.atom_type[atom_idx]
                func_idx = cluster.func[i]
                cluster_corr *= self.corr_func_table(func_idx, atom_type)
            
            if valid:
                total_corr += cluster_corr
                count += 1
        
        return total_corr / count if count > 0 else 0.0
```

## 3.5 SSOS搜索模块

### 3.5.1 模块概述

SSOS搜索模块基于随机化正交匹配追踪算法，从候选构型中筛选最优结构组合。

### 3.5.2 算法实现

```python
def randomized_omp(A, b, k):
    """
    随机化正交匹配追踪算法
    
    参数：
        A: 关联函数矩阵（转置后）
        b: 目标向量
        k: 稀疏度（选择的结构数）
    
    返回：
        w: 稀疏权重向量
    """
    best_residual = 1
    
    while best_residual > 1E-5:
        # 随机选择初始结构
        support = np.random.choice(A.shape[1], size=1, replace=False)
        residual = b.copy()
        
        for i in range(k-1):
            # 计算相关性
            correlations = A.T @ residual
            correlations[list(support)] = 0
            
            # 选择最大相关性（带随机性）
            max_indices = np.where(correlations == np.max(correlations))[0]
            new_index = np.random.choice(max_indices)
            support = np.append(support, new_index)
            
            # 非负最小二乘求解权重
            A_support = A[:, support]
            x_support, _ = nnls(A_support, b)
            
            # 更新残差
            residual = b - A_support @ x_support
        
        best_residual = np.linalg.norm(residual)
    
    # 构建完整权重向量
    best_x = np.zeros(A.shape[1])
    best_x[support] = x_support
    
    return best_x
```

# 第五章 总结与展望

## 5.1 工作总结

本论文围绕无序合金模拟的自动化和智能化需求，设计并实现了一套完整的软件系统。主要工作包括：

**（1）系统架构设计**

设计了模块化的四层架构，包括用户接口层、应用层、核心算法层和基础设施层。系统划分为超胞构建、不可约构型生成、关联函数计算和SSOS搜索四个核心模块，各模块功能独立、接口清晰。

**（2）超胞构建模块实现**

实现了基于对称性分析和Delaunay三角化优化的超胞构建算法。算法能够自动生成所有可能的超胞结构，并消除对称等价结构，显著减少了后续计算量。

**（3）不可约构型生成模块实现**

利用空间群对称性，开发了高效的不可约构型生成方法。该方法能够消除对称等价构型，对于高对称性晶体可将构型数减少几个数量级。

**（4）关联函数计算模块实现**

基于团簇展开理论，使用Python实现了关联函数的计算模块。模块支持多种晶格类型（FCC、BCC、HCP等）和多元合金体系，代码简洁高效。

**（5）SSOS搜索模块实现**

研究并实现了随机化正交匹配追踪算法，开发了基于Python的SSOS搜索模块。模块能够从大量候选构型中筛选出最优结构组合，支持用户指定结构数量。

**（6）系统集成与验证**

将四个模块集成为完整的工作流程，以Cu-Au二元合金为例进行了系统验证。结果表明系统能够正确生成超胞、计算关联函数并搜索得到合理的SSOS结构集合。

## 5.2 创新点

本论文的主要创新点如下：

**（1）完整的自动化工作流程**

提出了从原胞到SSOS的完整自动化工作流程，实现了各模块的无缝衔接，用户只需提供初始结构和参数即可获得最终结果。

**（2）超胞生成的对称性去重**

在超胞生成过程中引入对称性分析，自动消除等价超胞，避免了重复计算，提高了计算效率。

开发了基于Python的SSOS搜索模块，代码简洁、易于理解和扩展，为后续集成到AI编程工具奠定了基础。
