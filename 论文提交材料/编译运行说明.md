# 代码编译与运行说明

## 一、环境要求

### 1. Python环境
- Python 3.8+
- 依赖包：numpy, scipy, matplotlib

```bash
pip install numpy scipy matplotlib
```

### 2. Fortran编译器
- **推荐**: GFortran (GNU Fortran编译器)
- **下载地址**: 
  - Windows: https://www.msys2.org/ (安装后运行 `pacman -S mingw-w64-x86_64-gcc-fortran`)
  - 或使用 MSYS2: `pacman -S mingw-w64-x86_64-gfortran`

---

## 二、编译Fortran代码

### Windows (MSYS2/GFortran)
```bash
# 进入fortran目录
cd 论文提交材料/代码/fortran

# 编译所有文件
gfortran -c supercell.f90 -o supercell.o
gfortran -c disorder.f90 -o disorder.o
gfortran -c configurations.f90 -o configurations.o
gfortran -c symmetry.f90 -o symmetry.o
gfortran -c structure.f90 -o structure.o
gfortran -c groups.f90 -o groups.o
gfortran -c functions.f90 -o functions.o
gfortran -c outfiles.f90 -o outfiles.o
gfortran -c progress.f90 -o progress.o
gfortran -c stdout.f90 -o stdout.o

# 链接生成可执行文件
gfortran *.o -o supercell.exe
```

### Linux/macOS
```bash
cd 论文提交材料/代码/fortran
gfortran *.f90 -o supercell
```
