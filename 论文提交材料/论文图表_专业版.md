## 图2：工作流程图 (Mermaid)

```mermaid
flowchart TD
    START[输入: POSCAR文件<br/>Cu-Au FCC结构] --> STEP1
    
    subgraph STEP1 [Step 1: 超胞构建]
        S1A[读取POSCAR]
        S1B[质因数分解 N=4]
        S1C[枚举变换矩阵]
        S1D[对称性去重]
        S1E[Delaunay优化]
        S1A --> S1B --> S1C --> S1D --> S1E
    end
    
    STEP1 --> OUT1[输出: 2个超胞<br/>SPOSCAR-1, SPOSCAR-2]
    OUT1 --> STEP2
    
    subgraph STEP2 [Step 2: 构型生成]
        S2A[读取超胞结构]
        S2B[获取空间群对称操作]
        S2C[枚举原子占位]
        S2D[对称性去重]
        S2A --> S2B --> S2C --> S2D
    end
    
    STEP2 --> OUT2[输出: 5个不可约构型<br/>CONFIG-1 ~ CONFIG-5]
    OUT2 --> STEP3
    
    subgraph STEP3 [Step 3: 关联函数计算]
        S3A[定义团簇集合]
        S3B[生成三角函数基组]
        S3C[遍历所有构型]
        S3D[计算关联函数矩阵]
        S3A --> S3B --> S3C --> S3D
    end
    
    STEP3 --> OUT3[输出: 5×4 关联函数矩阵]
    OUT3 --> STEP4
    
    subgraph STEP4 [Step 4: SSOS搜索]
        S4A[构建目标向量]
        S4B[随机化OMP算法]
        S4C[非负最小二乘]
        S4D[输出最优组合]
        S4A --> S4B --> S4C --> S4D
    end
    
    STEP4 --> END[输出: SSOS结构集<br/>k=2, 残差=0.000000]
```
