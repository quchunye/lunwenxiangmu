# 项目使用快速指南

**版本**: 1.0  
**更新日期**: 2026-03-01

## 🚀 快速开始

### 1. 环境配置

```bash
# 配置 Fortran 环境
fortran_env.bat

# 配置 Python 环境
python_env.bat
```

### 2. 编译和运行 Fortran 程序

```bash
# 编译
gfortran hello_fortran.f90 -o hello_fortran.exe

# 运行
.\hello_fortran.exe
```

### 3. 转换为 Python

```bash
# 运行转换脚本
python src\python\fortran_to_python_converter.py

# 运行转换后的 Python 程序
python src\python\hello_fortran_converted.py
```

### 4. 备份

```bash
# 运行自动备份
scripts\backup\run_backup.bat
```

## 📊 备份策略

### 三层备份
1. **Git 版本控制** - 每次提交
2. **本地时间戳备份** - 自动创建
3. **阿里云备份** - ZIP 压缩包

### 备份位置
- 本地：`backups/timestamped/backup_YYYYMMDD_HHMMSS/`
- 阿里云：`backups/aliyun/aliyun_backup_YYYYMMDD_HHMMSS.zip`

### 恢复方法
```bash
# 从 Git 恢复
git checkout <commit-hash>

# 从备份恢复
copy backups\timestamped\backup_YYYYMMDD_HHMMSS\* . /Y
```
