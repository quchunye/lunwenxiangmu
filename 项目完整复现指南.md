## 四、关键概念解释（答辩用）

### 4.1 四个核心过程

| 过程 | 输入 | 输出 | 关键理解 |
|------|------|------|---------|
| 1. 超胞构建 | 纯Cu结构(1原子) | 超胞(4原子位点) | 只需要保存的内容
```

### 4.1 输入文件解释
```
FCC Cu              ← 晶格类型注释
 4.0                ← 晶格常数 (Å)
 0.0 0.5 0.5        ← 晶格向量a (FCC特征)
 0.5 0.0 0.5        ← 晶格向量b (FCC特征)
 0.5 0.5 0.0        ← 晶格向量c (FCC特征)
 Cu                 ← 元素类型 (定义晶格框架)
 1                  ← 原子数
 Direct             ← 分数坐标
 0.0 0.0 0.0        ← 原子位置
```

**重要说明**: 
- 输入纯Cu结构只定义**晶格框架**
- 合金成分在**构型生成阶段**引入
- 程序自动生成Cu/Au组合

### 4.2 四个核心过程

| 过程 | 输入 | 输出 | 合金信息 |
|------|------|------|---------|
| 1. 超胞构建 | 纯Cu结构 | 4原子超胞 | ❌ 还没有 |
| 2. 构型生成 | 超胞+Cu/Au类型 | 5个不可约构型 | ✅ 这里引入 |
| 3. 关联函数 | 构型列表 | 关联函数矩阵 | ✅ 已有 |
| 4. SSOS搜索 | 矩阵+目标成分 | 选中构型+权重 | ✅ 指定成分 |

## 六、答辩常见问题

### Q1: 为什么要用纯Cu结构作为输入？
> 输入的纯Cu结构只是定义了晶格框架（FCC结构），还没有涉及合金成分。合金成分是在构型生成阶段引入的。

### Q2: 关联函数有什么物理意义？
> 关联函数描述原子的排列特征。点团簇关联函数Π_0反映成分，双点团簇关联函数反映最近邻原子对的关联。

### Q3: SSOS和SQS有什么区别？
> SQS输出一个结构，只能近似模拟无序状态。SSOS输出多个结构的组合，精度可调节，权重有明确物理意义。
